# Procedural Planets
Procedural tectonic planets

## Dependencies
- QT5
- libQGLViewer
- CGAL

## Compilation

with `qmake`

```
cd src
qmake
make -j
```

or with `cmake`
```
cd src
mkdir build && cd build
cmake ..
make -j
```
