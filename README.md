# Procedural Planets
Procedural tectonic planets

## Dependencies
- [QT5](https://www.qt.io)
- [libQGLViewer](https://libqglviewer.com)
- [CGAL](https://www.cgal.org)
- C++ 20 compiler

## Compilation

with `qmake`

```
cd src
qmake
make -j
```

or with `cmake`
```
cd src
mkdir build && cd build
cmake ..
make -j
```

To build the documentation, use `doxygen` in the root directory.
